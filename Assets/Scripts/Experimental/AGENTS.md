# AGENTS.md

## Project: Into the Void
Build a lightweight browser game with an atmospheric sci-fi void theme.

## Implementation rules
- Prefer plain HTML, CSS, and JavaScript unless the repo already uses a frontend framework.
- Keep the first version small: one playable loop, clear controls, score/progress UI, and restart flow.
- Use placeholder or generated-style assets; do not add paid or licensed assets.
- Make the game responsive for desktop and mobile-sized screens.

## Codex workflow
1. Draft a short game design brief.
2. Implement the smallest playable version.
3. Run available tests or a local browser smoke test.
4. Iterate on controls, readability, and game feel.
